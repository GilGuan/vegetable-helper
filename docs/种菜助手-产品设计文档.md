# 种菜助手 - 产品设计文档

## 一、信息架构

```
种菜助手
│
├── 首页（Tab 1）
│   ├── 定位栏（当前省份）
│   ├── 当前月份推荐种植
│   └── 快速入口
│       ├── 种植日历
│       ├── 我的菜园
│       └── 种植指南
│
├── 种植日历（Tab 2）
│   ├── 月份选择器（1-12月横向滑动）
│   ├── 省份选择器
│   ├── 作物分类筛选（全部/叶菜/瓜果/豆类/根茎/葱蒜）
│   └── 作物卡片列表
│       ├── 作物图标
│       ├── 作物名称
│       ├── 适宜种植月份
│       └── 种植难度
│
├── 我的菜园（Tab 3）
│   ├── 已种植作物列表
│   │   ├── 作物名称
│   │   ├── 种植日期
│   │   ├── 生长天数
│   │   └── 操作按钮（查看详情/记录打理）
│   └── 添加种植按钮
│
├── 作物详情页
│   ├── 作物基本信息
│   │   ├── 名称、分类
│   │   ├── 生长周期
│   │   ├── 种植难度
│   │   └── 适宜省份和月份
│   ├── 种植步骤
│   │   └── 时间轴形式展示
│   ├── 注意事项
│   └── 操作按钮
│       ├── 开始种植
│       └── 添加到待种清单
│
├── 打理记录页
│   ├── 作物信息卡
│   ├── 打理记录列表
│   │   ├── 日期
│   │   ├── 类型（浇水/施肥/除虫/采收）
│   │   └── 备注
│   └── 添加记录按钮
│       ├── 类型选择
│       ├── 备注输入
│       └── 保存
│
└── 我的（Tab 4）
    ├── 种植统计
    │   ├── 已种植数量
    │   ├── 收获数量
    │   └── 打理记录数
    ├── 种植历史
    ├── 收获记录
    └── 设置
        ├── 切换省份
        └── 清除数据
```

## 二、页面流程

### 2.1 核心流程

```
用户打开小程序
    │
    ▼
自动定位省份 → 展示当前月份推荐作物
    │
    ├─── 点击作物 → 查看种植指南 → 开始种植 → 添加到"我的菜园"
    │
    ├─── 切换月份 → 查看其他月份推荐作物
    │
    └─── 切换省份 → 查看其他省份推荐作物
```

### 2.2 打理记录流程

```
进入"我的菜园"
    │
    ▼
选择已种植作物
    │
    ▼
点击"记录打理"
    │
    ▼
选择打理类型（浇水/施肥/除虫/采收）
    │
    ├─── 填写备注（可选）
    │
    ▼
保存 → 展示在打理记录列表
```

## 四、交互设计

### 4.1 省份切换

**触发**：点击首页/日历页的定位栏

**交互**：
1. 弹出省份选择器（ picker ）
2. 用户选择省份
3. 自动刷新作物列表

**默认行为**：
- 首次打开：自动获取位置（需授权）
- 拒绝授权：默认显示"广东省"

---

### 4.2 月份切换

**触发**：在种植日历页滑动月份选择器

**交互**：
1. 横向滑动选择月份（1-12月）
2. 当前月份高亮
3. 自动刷新作物列表

**默认行为**：
- 默认选中当前月份

---

### 4.3 作物卡片点击

**触发**：点击作物卡片

**交互**：
1. 跳转到作物详情页
2. 显示完整种植指南
3. 底部显示"开始种植"按钮

---

### 4.4 开始种植

**触发**：点击"开始种植"按钮

**交互**：
1. 弹出日期选择器（默认今天）
2. 确认后添加到"我的菜园"
3. 自动创建一条"播种"记录
4. Toast 提示"已添加到我的菜园"

---

### 4.5 记录打理

**触发**：点击"记录打理"按钮

**交互**：
1. 弹出打理类型选择（浇水/施肥/除虫/采收）
2. 可选填写备注
3. 保存后展示在打理记录列表
4. Toast 提示"记录成功"

## 五、视觉规范

### 5.1 配色方案

| 用途 | 颜色 | 色值 |
|------|------|------|
| 主色 | 绿色 | #4CAF50 |
| 辅色 | 浅绿 | #81C784 |
| 背景 | 米白 | #FFF8E1 |
| 文字 | 深灰 | #333333 |
| 次要文字 | 灰色 | #999999 |

### 5.2 字体规范

| 用途 | 字号 | 字重 |
|------|------|------|
| 页面标题 | 18px | Bold |
| 卡片标题 | 16px | Medium |
| 正文 | 14px | Regular |
| 辅助文字 | 12px | Regular |

### 5.3 图标风格

- 使用圆形背景 + 中心图标
- 作物图标：卡通风格，色彩鲜明
- 操作图标：线性图标，简洁

## 六、异常处理

### 6.1 定位失败

**场景**：用户拒绝授权或定位失败

**处理**：
- 默认显示"广东省"
- 用户可手动切换省份

**场景**：我的菜园为空
