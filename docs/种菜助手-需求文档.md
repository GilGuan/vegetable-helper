# 种菜助手 - 产品需求文档（PRD）

## 一、产品概述

### 1.1 产品定位

**种菜助手**是一款面向个人用户的微信小程序，帮助用户解决"什么时节种什么、怎么种、怎么打理"的问题。

### 1.2 目标用户

- **核心用户**：项目发起人（关毅）
- **用户画像**：租了块地准备种菜，但缺乏种植知识的新手
- **使用场景**：
  - 规划种植计划时，查看什么时节适合种什么
  - 种植过程中，查看种植步骤和注意事项
  - 日常打理时，接收浇水、施肥、除虫提醒

### 1.3 核心价值

1. **降低种植门槛**：新手也能知道什么时候种什么
2. **系统化知识**：从播种到收获的完整指南
3. **智能提醒**：不忘记浇水、施肥、除虫

## 二、功能需求

### 2.1 功能架构

```
种菜助手
├── 首页
│   ├── 当前节气推荐种植
│   ├── 我的菜园（已种植列表）
│   └── 快速添加种植
├── 种植日历
│   ├── 按月份查看
│   ├── 按节气查看
│   └── 按作物筛选
├── 种植指南
│   ├── 作物详情
│   ├── 种植步骤
│   └── 注意事项
├── 日常打理
│   ├── 今日任务
│   ├── 任务提醒
│   └── 任务记录
└── 我的
    ├── 种植记录
    ├── 收获记录
    └── 设置
```

### 2.2 核心功能详解

#### 功能1：种植日历

**功能描述**：按月份展示适合种植的作物，支持按地区筛选

**页面元素**：
- 地区选择器（省份/城市）
- 月份选择器（1-12月，横向滑动）
- 当前月份适合种植的作物列表（卡片形式）
- 作物分类筛选（叶菜类、根茎类、瓜果类、豆类等）

**地区判断逻辑**：
- 首次打开：自动获取用户位置（需授权），定位到省份
- 手动选择：支持切换省份
- 不同省份展示不同的作物推荐

**MVP 作物清单（广东省，共30种）**：

| 分类 | 作物 |
|------|------|
| 叶菜类 | 生菜、小白菜、菠菜、菜心、芥蓝、空心菜、茼蒿、香菜、油麦菜、韭菜 |
| 瓜果类 | 番茄、黄瓜、辣椒、茄子、苦瓜、丝瓜、冬瓜、南瓜 |
| 豆类 | 豆角、四季豆、豌豆 |
| 根茎类 | 萝卜、胡萝卜、土豆、芋头 |
| 葱蒜类 | 葱、蒜、姜 |

**数据结构**：
```json
{
  "crop": {
    "id": "tomato_001",
    "name": "番茄",
    "category": "瓜果类",
    "suitableProvinces": ["广东省", "广西壮族自治区", "福建省", "海南省"],
    "suitableMonths": {
      "广东省": [2, 3, 4, 5, 6, 7, 8, 9, 10],
      "福建省": [3, 4, 5, 6, 7, 8, 9]
    },
    "growthCycle": 90,
    "difficulty": "中等"
  }
}
```

**交互逻辑**：
1. 用户打开小程序，自动获取位置，确定地区
2. 默认展示当前月份适合种植的作物
3. 用户可切换月份查看其他月份适合种的作物
4. 点击作物卡片，跳转到种植指南详情页

---

#### 功能2：种植指南

**功能描述**：告诉用户怎么种

**页面元素**：
- 作物基本信息（名称、分类、生长周期、难度）
- 种植步骤（时间轴形式）
- 注意事项（tips 列表）
- 常见问题（FAQ）
- "开始种植"按钮

**数据结构**：
```json
{
  "guide": {
    "cropId": "tomato_001",
    "steps": [
      {
        "order": 1,
        "title": "播种",
        "description": "将种子浸泡4-6小时后播种",
        "duration": 7,
        "tips": ["保持土壤湿润", "避免阳光直射"]
      },
      {
        "order": 2,
        "title": "移栽",
        "description": "幼苗长出4-5片真叶时移栽",
        "duration": 14,
        "tips": ["选择阴天或傍晚移栽", "浇透定根水"]
      }
    ],
    "faqs": [
      {
        "question": "番茄需要搭架子吗？",
        "answer": "需要，番茄是藤本植物，需要搭架子或绑绳支撑。"
      }
    ]
  }
}
```

**交互逻辑**：
1. 用户浏览种植指南
2. 点击"开始种植"，添加到"我的菜园"
3. 自动生成任务提醒（根据生长周期）

---

#### 功能3：日常打理

**功能描述**：记录浇水、施肥等日常打理动作

**页面元素**：
- 我的菜园列表（已种植作物）
- 每个作物的打理记录入口
- 打理记录列表（浇水、施肥、除虫、采收等）
- 快速记录按钮

**数据结构**：
```json
{
  "careRecord": {
    "id": "record_001",
    "plantingId": "planting_001",
    "cropId": "tomato_001",
    "type": "watering",  // watering, fertilizing, pest_control, harvest
    "typeName": "浇水",
    "note": "今天浇了两次水",
    "createdAt": "2026-03-27T08:00:00Z"
  }
}
```

**交互逻辑**：
1. 用户在"我的菜园"选择某个作物
2. 点击"记录打理"按钮
3. 选择打理类型（浇水/施肥/除虫/采收）
4. 可选填写备注
5. 保存记录，展示在打理记录列表

---

**页面元素**：

```json
    "harvest": {
      "date": "2026-06-25",
      "yield": "5kg",
      "note": "收成不错，下次可以多种几棵"
    }
  }

```

## 三、非功能需求

### 3.1 性能要求

- 小程序启动时间 < 2秒
- 页面加载时间 < 1秒
- 支持离线查看已缓存数据

### 3.2 数据存储

- 用户数据存储：微信云开发（CloudBase）
- 作物数据：云数据库
- 用户上传图片：云存储

### 3.3 兼容性

- 微信小程序基础库版本：2.20.0+
- 支持 iOS 和 Android

## 四、数据来源方案

### 4.1 节气数据

**来源**：和风天气 API（免费额度足够）

**接口**：`https://dev.qweather.com/v7/astronomy/solar-terms`

**调用频率**：每月调用1次，获取节气时间表

### 4.2 作物种植知识

**来源**：AI 生成 + 人工校验

**生成方式**：
1. 使用大模型生成常见作物的种植知识
2. 项目发起人人工校验准确性
3. 存入云数据库

**Prompt 示例**：
```
请生成番茄的种植指南，包括：
1. 基本信息（分类、生长周期、种植难度）
2. 适宜种植的月份和节气
3. 种植步骤（播种、移栽、管理等）
4. 注意事项（浇水、施肥、病虫害防治）
5. 常见问题（3-5个）

输出为 JSON 格式。
```

### 4.3 用户自建数据

支持用户添加自定义作物和种植知识。

## 五、技术架构

### 5.1 整体架构

```
┌─────────────────────────────────────┐
│           微信小程序前端              │
│  (WXML + WXSS + JavaScript)         │
└──────────────┬──────────────────────┘
               │
┌──────────────▼──────────────────────┐
│          微信云开发后端              │
│  ┌─────────┬─────────┬─────────┐    │
│  │云数据库  │ 云存储   │ 云函数  │    │
│  └─────────┴─────────┴─────────┘    │
└─────────────────────────────────────┘
```

### 5.2 技术选型

| 模块 | 技术选型 | 理由 |
|------|---------|------|
| 前端框架 | 微信小程序原生 | 简单、够用 |
| 后端服务 | 微信云开发 | 免运维、免服务器成本 |
| 数据库 | 云数据库 | 开箱即用 |
| 文件存储 | 云存储 | 存储用户上传的照片 |
| 定时任务 | 云函数定时触发 | 每日生成任务提醒 |
